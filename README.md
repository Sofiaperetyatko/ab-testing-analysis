# ab-testing-analysis
В данном проекте проводится A/B тест. В датасете представлены 2 группы: control_group и test_group. Они знакомятся с двумя немного разными версиями одного и того же приложения/сайта/продукта. Пользователи из контрольной группы знакомятся с изначальной версией продукта, а пользователи из тестовой группы — с изменённой версией продукта. Цель A/B-тестирования — выяснить, влияет ли знакомство с тестовой версией по сравнению с оригинальной версией на поведение пользователей с точки зрения количества покупок, кликов на сайт, показов рекламы.

Прежде чем это выяснить был проведен предварительный анализ данных: данные привели к одному типу, убрали выбросы, посчитали маркетинговые метрики. Далее был подобран оптимальный статистический тест для проверки на статистическую значимость в изменении метрик. Согласно результатам теста Манна-Уитни новая версия не дала значимых изменений в метриках.
